What you are looking at

This dashboard benchmarks the input layer of a spiking neural network (SNN) designed to discriminate real neuronal action potentials from noise artefacts in extracellular electrode recordings.

The core question: Given a 64-sample (≈2.1 ms at 30 kHz) voltage snippet captured at a threshold crossing, is this a real action potential from a neuron, or electrical noise?

The pipeline extracts numerical features from each snippet (e.g. trough depth, spike width, baseline noise level), benchmarks how well each feature separates the two classes, then evaluates several classifiers on combinations of features. The hand-coded WeightBank input layer uses quantile-initialised Gaussian population-code neurons so that real spikes produce high activation and noise produces low activation — without any gradient-based training.

Feature glossary

Features are grouped by tier. Tier 1/2 features are causal (use only past samples) and suitable for real-time streaming pipelines. Tier 3 (prefix ev_) features require the complete snippet and add one snippet-length latency.
